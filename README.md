# Лабораторная работа №13-14
Коллекции, обобщения и функциональный стиль в Kotlin
## Описание
Данная лабораторная работа посвящена изучению продвинутых возможностей языка Kotlin, которые активно используются при разработке Android-приложений.
В рамках работы рассматриваются:
- обобщённые типы (Generics);
- коллекции Kotlin;
- функции высшего порядка;
- extension-функции;

Все примеры ориентированы на практическое применение и подготовку к разработкемобильных приложений.

## Содержание работы
1. **Обобщённые типы (Generics)**
    - Создание универсального класса `Question<T>` для хранения вопросов с разными типами ответов.
    - Пример использования: `Question<String>`, `Question<Boolean>`, `Question<Int>`.
2. **Использование enum class**
    - Создание перечисления `Difficulty` для ограничения возможных значений сложности вопроса.
    - Преимущества: защита от опечаток, контроль значений на этапе компиляции.
3. **Data class**
    - Преобразование класса `Question` в `data class`.
    - Автоматическая генерация методов `equals()`, `hashCode()`, `toString()`, `copy()`.
4. **Singleton и companion object**
    - Создание объекта `StudentProgress` для отслеживания прогресса.
    - Использование `companion object` внутри класса `Quiz`.
5. **Extension-функции и свойства**
    - Добавление свойства `progressText` и функции `printProgressBar()` к `StudentProgress` без изменения исходного класса.
    - Пример вызова: `Quiz.printProgressBar()`.
6. **Интерфейсы**
    - Создание интерфейса `ProgressPrintable`.
    - Реализация интерфейса в классе `Quiz`.
7. **Scope-функции**
    - Использование `let()` для устранения повторений.
    - Использование `apply()` для инициализации объектов.
8. **Коллекции в Kotlin**
    - **Массивы (Array)**: создание, доступ по индексу, изменение элементов.
    - **Списки (List, MutableList)**: `listOf()`, `mutableListOf()`, добавление, удаление, перебор.
    - **Множества (Set)**: уникальность элементов, быстрый поиск.
    - **Словари (Map)**: хранение пар ключ-значение.
9. **Функции высшего порядка с коллекциями**
    - `forEach()` — перебор элементов.
    - `map()` — преобразование коллекции.
    - `filter()` — фильтрация по условию.
    - `groupBy()` — группировка по ключу.
    - `fold()` — аккумуляция значений.
    - `sortedBy()` — сортировка.

## Примеры кода

#### Обобщённый класс вопроса:
```kotlin
data class Question<T>(
    val questionText: String,
    val answer: T,
    val difficulty: Difficulty
)
```
#### Enum:
```kotlin
enum class Difficulty {
    EASY, 
    MEDIUM, 
    HARD
}
```
#### Использование map и filter:
```kotlin
val softCookies = cookies.filter { it.softBaked }
val menuText = cookies.map { "${it.name} - $${it.price}" }
```
#### Scope-функция:
```kotlin
question2.let {
  println(it.questionText)
  println(it.answer)
  println(it.difficulty)
}
```
